# Day 04 – Data Cleaning

## Why Data Cleaning Matters

Real-world datasets are messy.

Common problems:

- Missing values
- Duplicate records
- Wrong data types
- Outliers
- Inconsistent formatting

Data cleaning improves reliability and accuracy.

## Important Pandas Functions

```python
drop_duplicates()
isnull()
fillna()
dropna()
```

## Policy Research Example

Before analysing defence budgets, researchers must verify:

- Missing observations
- Duplicate country records
- Incorrect values
- Inconsistent units

Clean data produces trustworthy analysis.
