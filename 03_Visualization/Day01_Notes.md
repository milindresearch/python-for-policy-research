# Day 01 - Matplotlib Basics

## What is Matplotlib?

Matplotlib is Python's most widely used data visualization library.

It helps convert data into:

- Line charts
- Bar charts
- Histograms
- Scatter plots
- Policy dashboards

---

## Why It Matters

Policy analysts rarely present raw tables.

Decision makers prefer visual insights.

Example:

Instead of:

India = 75

USA = 916

China = 296

A chart immediately reveals relative differences.

---

## Core Commands

```python
import matplotlib.pyplot as plt
```

Creates plotting environment.

```python
plt.bar()
```

Creates bar chart.

```python
plt.title()
```

Adds chart title.

```python
plt.xlabel()
```

Adds x-axis label.

```python
plt.ylabel()
```

Adds y-axis label.

```python
plt.show()
```

Displays chart.
