# Day 02 - Line Charts

## What is a Line Chart?

A line chart shows how a variable changes over time.

It is one of the most important charts for policy research.

---

## Why Analysts Use It

Line charts help identify:

- Trends
- Growth
- Decline
- Cycles
- Turning points

---

## Core Command

```python
plt.plot(x, y)
```

Creates a line chart.

---

## Example Uses

- Defence budget trends
- GDP growth
- Inflation rates
- Trade flows
- Population growth

---

## Real Policy Application

Instead of comparing one year only, analysts examine trends across multiple years.

For example:

India Defence Spending

2020 → 66

2021 → 70

2022 → 72

2023 → 74

2024 → 75

A line chart immediately shows the long-term direction.

---

## Key Takeaway

Bar charts compare categories.

Line charts analyze change over time.
