# Day 01 – Variables and Data Types

## Concept

Variables are used to store data in Python.

Common data types:

- Integer (int)
- Float (float)
- String (str)
- Boolean (bool)

## Example

```python
# Illustrative example only

country = "Example Country"
defence_budget = 500000
growth_rate = 6.5
is_nuclear_power = True

print(country)
print(defence_budget)
```
